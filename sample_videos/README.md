# Sample Videos for Testing

To ensure the best experience when testing the AuthentiKit prototype, please use the sample videos provided in this folder. 

**Why use these videos?**
- They have been optimized (compressed to < 5MB) to run quickly and flawlessly within the Google Cloud Video Intelligence and Gemini 1.5 Pro inline Base64 payload limits.
- They are selected to specifically showcase the physics engine, shadow detection, and VAR text extraction capabilities.

### Adding Your Own Videos
If you wish to test your own videos, please ensure:
1. The file size is **under 5MB**.
2. The duration is short (under 10 seconds).
3. The video clearly shows a sports play or sports context (scoreboards, etc.).
